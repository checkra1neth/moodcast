# 🎧 MoodCast — AI Audio Stories That Match Your Mood

> Pick a mood. Get a cinematic audio experience — voice, music, and sound effects — all generated by AI in real time.

![MoodCast](https://img.shields.io/badge/ElevenLabs-Hackathon-6ee591?style=for-the-badge) ![Node.js](https://img.shields.io/badge/Node.js-22+-339933?style=for-the-badge&logo=node.js&logoColor=white)

## The Problem

You want to relax, focus, get energized, or fall asleep — but finding the right audio content means scrolling through endless playlists, podcasts, and apps. Nothing is personalized. Nothing combines voice, music, and atmosphere into one seamless experience.

## The Solution

MoodCast generates **personalized cinematic audio stories** on demand. Choose a mood, and AI creates:

- 🎙️ **Voice narration** with emotional delivery (ElevenLabs TTS v3 + Audio Tags)
- 🎵 **Original music** that matches the scene (ElevenLabs Music API)
- 🎬 **Cinematic sound effects** synced to story moments (ElevenLabs Sound Effects API)
- ✨ **AI-written stories** from custom prompts in any language (ElevenLabs Conversational AI)

All four layers generate **in parallel** and play together with a real-time mixer.

## How It Works

```
User picks mood → AI writes story + sound design → 4 audio layers generate in parallel → Cinematic playback with mixer
```

1. **Choose a mood**: Relax 🌿 / Focus 🎯 / Energy ⚡ / Sleep 🌙
2. **Optionally**: write a custom scene prompt, pick a voice, set duration (15s–10min)
3. **Generate**: voice + music + SFX cues all render simultaneously via SSE streaming
4. **Play**: layered audio with per-channel volume control, seek, loop

## ElevenLabs APIs Used

| API | What it does in MoodCast |
|-----|--------------------------|
| **Text-to-Speech v3** | Narrates stories with Audio Tags for emotional delivery (`[whispering]`, `[excited]`, `[calm]`) |
| **Music** | Composes original instrumental tracks matched to the story's scene and mood |
| **Sound Effects** | Generates cinematic SFX cues (footsteps, rain, thunder, etc.) timed to story moments |
| **Conversational AI** | Powers the story-writing agent that creates narratives with precise word counts and SFX choreography |

## What Makes It Special

- **4 APIs in one experience** — not just TTS, but a full cinematic audio pipeline
- **Parallel generation** — voice, music, and SFX generate simultaneously with live progress
- **Cinematic SFX sync** — sound effects are timed to specific story moments (e.g., "door creaks" when the story says "you open the door")
- **Audio Tags** — emotional delivery control (`[WHISPERING]`, `[EXCITED]`) for natural narration
- **Any language** — custom prompts auto-detect language; stories generate in Ukrainian, Spanish, Japanese, etc.
- **Real-time mixer** — adjust voice/music/SFX balance during playback

## Quick Start

```bash
# Clone
git clone https://github.com/checkra1neth/moodcast.git
cd moodcast

# Install
npm install

# Configure
cp .env.example .env
# Add your ElevenLabs API key to .env

# Run
npm start
# Open http://localhost:3000
```

## Environment Variables

| Variable | Required | Description |
|----------|----------|-------------|
| `ELEVENLABS_API_KEY` | Yes | Your ElevenLabs API key |
| `ELEVENLABS_AGENT_ID` | No | Cached Conversational AI agent ID (auto-created if missing) |

## Tech Stack

- **Backend**: Node.js + Express
- **Frontend**: Vanilla JS, CSS (glassmorphism UI)
- **AI**: ElevenLabs TTS v3, Music, Sound Effects, Conversational AI
- **Streaming**: Server-Sent Events for real-time generation progress

## Project Structure

```
├── server.mjs        # Express API — generation endpoint with SSE
├── enhance.mjs       # Conversational AI agent for story writing
├── stories.mjs       # Pre-written story library (4 moods × 3 durations)
├── moods.mjs         # Mood configs, voice settings, duration presets
├── public/
│   ├── index.html    # Three-screen SPA (select → generate → play)
│   ├── app.js        # Client logic, SSE handling, audio mixer
│   └── style.css     # Glassmorphism dark UI
```
